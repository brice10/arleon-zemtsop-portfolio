📚 The Research

📄

Master's Thesis

"Towards a Model-Driven Engineering Approach for Workflow Management in a Web Application"

Developing a web application framework that integrates process management using a novel operational process specification language.

MDE Workflow LSAWfP DSL Web Apps

My work focuses on developing a web application framework that integrates business process management using a novel operational process specification language called LSAWfP (Language for the Specification of Administrative Workflow Processes).

🔬 Key Contributions

Here are the main contributions of my research:

01

Software Architecture Proposal

Proposed a comprehensive software architecture for a web application creation system that includes integrated business process management capabilities.

02

LSAWfP4WEB Language

Introduced LSAWfP4WEB, a new language based on LSAWfP, specifically designed for specifying web-based business processes.

03

Domain Specific Embedded Language (DSEL)

Developed a JSON-based DSEL for creating, saving, and sharing workflow specifications in a portable and standardized format.
